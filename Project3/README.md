# Face Recognition Project

## 프로젝트 소개
OpenCV와 Dlib를 활용한 실시간 얼굴 인식 시스템입니다.

영상 및 이미지에서 얼굴을 탐지하고,
등록된 얼굴 데이터와 비교하여 인식합니다.

---

## 사용 기술

- C++
- OpenCV
- Dlib
- Multi Thread
- Visual Studio 2026

---

## 주요 기능

- 이미지 얼굴 인식
- 영상 얼굴 인식
- 실시간 얼굴 탐지
- FPS 출력
- 멀티스레드 기반 처리

---

## 실행 방법

1. 모델 파일 다운로드
2. known_faces 폴더 생성
3. 프로젝트 실행

---

## 모델 파일

아래 모델 파일이 필요합니다.

- shape_predictor_5_face_landmarks.dat
- dlib_face_recognition_resnet_model_v1.dat
